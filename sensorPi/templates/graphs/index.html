This is the home page for sensorPi. If you are an admin, log in at the top right corner to see the admin panel.

This project is a web application that displays data from a Raspberry Pi with SCD-40 sensor. The data, such as temperature, humidity and CO2 levels, is collected every ~5 seconds and stored in a database. The data is then displayed in graphs and stats on this website.

You can navigate to the graphs page to see the data for today. From there you could use hyperlinks to see the data for other days. Alternatively, just type add '/graphs/YYYY-MM-DD' to the URL to see the data for that day.
